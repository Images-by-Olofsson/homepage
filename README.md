## Features

Modern design with minimalist aesthetics and focus on imagery. Advanced animations including wave effects, 3D scroll transforms, and smoke particles. Fully responsive across all devices with optimized performance.

<br>

## Quick Start

**Prerequisites**

Node.js 18.0 or higher

**Installation**

```bash
git clone https://github.com/Images-by-Olofsson/homepage.git
cd homepage
npm install
```

**Development**

```bash
npm run dev
```

**Build**

```bash
npm run build
npm run preview
```
